PEC Drive Loader

Crear carpetas, escanear Drive y publicar datos para el panel GitHub Pages, dejando claro que esta superficie administra fuentes y no reemplaza el panel público ni el visor.
Atención
Esta pantalla es administrativa. Su función es preparar Drive, la Sheet índice y los endpoints de datos. No es la web pública de consulta ni el visor ejecutivo que se comparte con usuarios finales.
Abrir panel público Abrir visor publicado
Si abres la URL base del Web App, estás entrando al backend del loader. Para el visor compartido se usa la misma Web App con ?view=visor.
Capacidades LLM asociadas
El loader no ejecuta investigación por sí mismo, pero alimenta al panel público y al copiloto LLM con la estructura documental, el índice maestro y los endpoints vivos.
  • Contexto ordena carpeta raíz, subcarpetas y spreadsheet operativo.
  • Fuentes publica JSON/CSV para que el panel lea evidencia documental actualizada.
  • Copiloto LLM habilita síntesis, comparación tecnológica y borradores técnicos desde el panel.
  • Trazabilidad mantiene separadas la capa administrativa, la capa pública y el visor compartido.
