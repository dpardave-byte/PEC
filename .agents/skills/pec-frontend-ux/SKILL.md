# Skill: PEC Frontend UX

Reglas de interfaz:

- Mantener jerarquia visual clara.
- Los bloques raiz deben verse como elementos principales.
- Evitar que un bloque raiz parezca subnivel.
- Evitar superposicion de textos, tarjetas, botones o graficos.
- Mantener un diseno ejecutivo, simple y util para toma de decisiones.
- La informacion critica debe verse sin exigir demasiados clics.
- Las alertas deben ser visibles y comprensibles.
- Los formularios deben ser consistentes y faciles de usar.
- Validar que la interfaz funcione en pantalla amplia y pantalla reducida.

Para cambios en el visor PEC:

- Verificar alineacion de bloques 1, 2, 3 y 4.
- Verificar que Inicio de Efectividad del Prestamo este a la izquierda como raiz.
- Verificar que las fases no pierdan legibilidad.
- Verificar que el Gantt no se rompa visualmente.
